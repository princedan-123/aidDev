This is an open source portfolio project that aims to recommend public materials (videos, books, articles) to software developers. The recommended materials are tailored towards programming, coding, software development.

How to use the project

In the landing page of the project, there is a search box. Enter your search query in the search box, then select the type of material you want to be recommended from the drop down select menu. Please note that by default video is selected. Then click on the recommended button.

In order for you to get the best out of this project, you should take note of the following points.

  • The project is software oriented hence it was programed to recommend software engineering content.
  • Your search query should be specific and limited to coding or software development
  • Your keyword matters, the more descriptive it is the better the search result. For example what is function in python and how to use it. This search keyword is better compared to functions
  • At the moment the project just uses Google APIs to recommend matrials, the materials are not provided by the authors of the project.
  • It should also be noted that although Google APIs are free, it is quota based. Each time you enter a search query you are using from the allocated quota and if it is exceeded, you may get an error message.
